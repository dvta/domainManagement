# Domain management

## How run the project

- Clone the project
- Run `composer install`
- run `npm install`
- run `npm run dev'` (use nvm 18)
- Run `php artisan serve`

## how to open the project

- Open the browser
- Go to `http://localhost:8000/` (domains and emails management)
- Go to `http://localhost:8000/email` (email inbox and etc)
